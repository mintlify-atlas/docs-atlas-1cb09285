## Introduction

The platform implements authentication using [NextAuth.js](https://next-auth.js.org/) v5 with OAuth 2.0 providers. The system supports multiple OAuth providers and implements custom session management with JWT tokens.

## Authentication Flow

The authentication flow follows the OAuth 2.0 authorization code flow:

1. User initiates sign-in via OAuth provider (Google or GitHub)
2. User is redirected to provider's authentication page
3. After successful authentication, user is redirected back to the application
4. NextAuth processes the callback and creates a session
5. User data is stored in the database and JWT token is generated
6. Session information is available throughout the application

## Quick Start

### Check Available Providers

<CodeGroup>

```bash curl
curl http://localhost:3000/api/auth/providers
```

```javascript fetch
const providers = await fetch('http://localhost:3000/api/auth/providers')
  .then(res => res.json());
console.log(providers);
```

</CodeGroup>

### Initiate Sign In

```typescript
import { signIn } from '@/auth'

// Sign in with Google
await signIn('google')

// Sign in with GitHub
await signIn('github')
```

### Get Current Session

```typescript
import { auth } from '@/auth'

export default async function Page() {
  const session = await auth()
  
  if (!session?.user) {
    return <div>Not authenticated</div>
  }
  
  return <div>Welcome, {session.user.name}!</div>
}
```

### Sign Out

```typescript
import { signOut } from '@/auth'

// Sign out user
await signOut()
```

## Configuration

The NextAuth configuration is defined in `src/auth.ts:15`:

```typescript
export const { handlers, signIn, signOut, auth } = NextAuth({
  providers: [Github, Google({
    clientId: process.env.AUTH_GOOGLE_ID,
    clientSecret: process.env.AUTH_GOOGLE_SECRET,
    authorization: {
      params: {
        scope: "openid email profile https://www.googleapis.com/auth/calendar.events",
        access_type: "offline",
      }
    }
  })],
  session: {
    maxAge: 60 * 60 * 24 * 30, // 30 days
    updateAge: 43200 // 12 hours
  },
  jwt: {
    maxAge: 60 * 60 * 24 * 30 // 30 days
  },
  pages: {
    signIn: '/inicio',
    signOut: '',
    error: '/'
  }
})
```
